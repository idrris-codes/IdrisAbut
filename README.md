# Idris Official Digital Profile

Готовый mobile-first сайт-визитка Идриса Шарипова.

## Запуск
Откройте `index.html` в браузере.

## Фото
Положите своё деловое фото рядом с `index.html` и назовите файл:

`my-photo.jpg`

## Файлы
- `index.html` — структура сайта
- `style.css` — премиальный mobile-first дизайн и адаптация
- `script.js` — выбор языка и анимации

## Деплой
Можно загрузить на GitHub Pages, Railway, Netlify, Vercel или любой статический хостинг.
