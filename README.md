# Idris Official Premium Site

Готовый проект для Railway.

## Запуск локально

```bash
npm install
npm start
```

Сайт запустится на порту `8080`, если Railway не передаст свой `PORT`.

## Фото

Положи своё фото в папку `public` и назови файл:

```text
my-photo.jpg
```

## Структура

```text
idris_final_railway/
├── package.json
├── railway.json
├── Procfile
├── server.js
└── public/
    ├── index.html
    ├── style.css
    └── script.js
```
